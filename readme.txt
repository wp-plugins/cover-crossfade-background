== Description ==

Easily insert a cover background into your website. Select on per page/post basis on admin edit post screen.
When setting multiple backgrounds a crossFade will be used.

== Installation ==

This section describes how to install the plugin and get it working.

e.g.

1. Upload `/coverbg` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress
3. Goto edit post/page screen and select your background(s).
4. Enjoy your cover background

== Frequently Asked Questions ==

= Where can i select a background? =

On the edit post/page screen.

= What if dont set a background on each page? =

Pages without a background set will use the homepage background.
